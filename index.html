# Artista Título

Cómo usar esta herramienta

Paso 1: Prepara tu archivo CSV

Crea un archivo CSV con columnas para "Artista" y "Título". Cada fila debe contener una canción.

Ejemplo:

Artista,Título
Daft Punk,Get Lucky
The Weeknd,Blinding Lights
Bad Bunny,Dakiti

Paso 2: Sube tu archivo

Haz clic en "Seleccionar archivo" y escoge tu archivo CSV.

Paso 3: Configura tu playlist

Ingresa un nombre y descripción para tu playlist.

Paso 4: Autoriza YouTube Music

Haz clic en "Crear Playlist" y autoriza la aplicación para acceder a tu cuenta de YouTube Music.
